## Run Locally

Clone the project

```bash
  git clone https://github.com/PramudyaReynaldi/notes-app-backend-hapi.js.git
```

Go to the project directory

```bash
  cd notes-app-backend-hapi.js
```

Install dependencies

```bash
  npm install
```

Npm init

```bash
  npm init -y
```

Start the server

```bash
  npm run start:dev
```




## API Reference

#### Get all items

```bash
  GET /notes
```


#### Get item

```bash
  GET /notes/{id}
```


#### Post item

```bash
  POST /notes
```

#### Delete item

```bash
  Delete /notes/{id}
```

#### Update item

```bash
  PUT /notes/{id}
```
