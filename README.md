
# Notes App Backend Hapi




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
```




## API Reference

#### Get all items

```http
  GET /notes
```


#### Get item

```http
  GET /notes/{id}
```


#### Post item

```http
  POST /notes
```

#### Delete item

```http
  Delete /notes/{id}
```

#### Update item

```http
  PUT /notes/{id}
```
